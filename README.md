# JavaScript Entry

このリポジトリは、中学1年生向けに JavaScript を通じてエンジニア基礎スキルを育成するための教材置き場です。単に文法を覚えるだけでなく、手を動かしながら「作る・直す・共有する」の流れを学ぶことを目的にしています。

## 学習の進め方

- **序盤（導入）**: Code.org を使い、ブロックやビジュアル環境を通じてプログラミングの考え方を身につけます。
- **中盤以降（発展）**: JavaScript.info と MDN を参照し、JavaScript の文法・DOM・Webの基礎へ段階的に進みます。
- **実践**: 学んだ内容をこのリポジトリのテンプレートで再実装し、GitHub で提出します。

## GitHub運用の基本

この講座では、以下の開発フローを基本とします。

1. Issue を作成して課題を明確化する
2. Branch を切って実装する
3. Pull Request（PR）を作成する
4. Review を受けて改善・マージする


## 生徒用ブランチについて

- `student/dev` ブランチでは、講師向け設計資料（講座設計・カリキュラム設計）を含めない運用にしています。
- 生徒は `README.md` と `template/web-basic` を中心に課題へ取り組んでください。
