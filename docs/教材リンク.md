# 教材リンク集

このページでは、講座で使う学習サイトを段階別にまとめています。序盤は Code.org を中心に、操作しながら概念をつかむことを重視します。中盤以降は JavaScript.info と MDN を使い、実際の Web 開発で必要な知識へ接続します。

## 序盤（Code.org）

- [https://code.org/learn](https://code.org/learn)
  - 初学者向けの学習入口です。短いチュートリアルで、プログラミングの基本概念を体験できます。
- [https://code.org/curriculum/csd](https://code.org/curriculum/csd)
  - Computer Science Discoveries のカリキュラム一覧です。授業単位で順序立てて学習を進められます。
- [https://studio.code.org/tools/applab](https://studio.code.org/tools/applab)
  - App Lab の作成画面です。画面部品と JavaScript を使って、アプリ作成の流れを学べます。
- [https://studio.code.org/tools/weblab](https://studio.code.org/tools/weblab)
  - Web Lab の作成画面です。HTML/CSS を中心に、Webページ構造と見た目の基礎を練習できます。
- [https://studio.code.org/tools/gamelab](https://studio.code.org/tools/gamelab)
  - Game Lab の作成画面です。ゲーム制作を通じて、座標・条件分岐・繰り返しの理解を深められます。
- [https://support.code.org/hc/en-us/articles/115001595051-Finding-curriculum-and-lesson-plans](https://support.code.org/hc/en-us/articles/115001595051-Finding-curriculum-and-lesson-plans)
  - Code.org のカリキュラムと授業案を探すための公式ガイドです。授業準備や進行計画の確認に使えます。
- [https://support.code.org/hc/en-us/articles/360039984911-What-are-the-differences-between-App-Lab-Game-Lab-Web-Lab-Sprite-Lab-and-Java-Lab-](https://support.code.org/hc/en-us/articles/360039984911-What-are-the-differences-between-App-Lab-Game-Lab-Web-Lab-Sprite-Lab-and-Java-Lab-)
  - 各 Lab の違いを説明した公式記事です。学習目的に合ったツールを選ぶ判断材料になります。

## 中盤以降（JavaScript.info / MDN）

- [https://javascript.info/](https://javascript.info/)
  - JavaScript の文法と考え方を体系的に学べる教材です。例題を通して、実装で使える知識を整理できます。
- [https://developer.mozilla.org/ja/docs/Web/JavaScript](https://developer.mozilla.org/ja/docs/Web/JavaScript)
  - MDN の JavaScript リファレンスです。構文や API を正確に確認する辞書として活用できます。
- [https://developer.mozilla.org/ja/docs/Web/API/Document_Object_Model](https://developer.mozilla.org/ja/docs/Web/API/Document_Object_Model)
  - DOM の基礎を学ぶためのページです。JavaScript で画面要素を操作する方法の理解に役立ちます。
- [https://developer.mozilla.org/ja/docs/Learn](https://developer.mozilla.org/ja/docs/Learn)
  - Web開発全体を学ぶ総合入口です。HTML/CSS/JavaScript を横断しながら、実践的に理解を広げられます。

## 教材の使い方（このリポジトリとの接続）

- Code.org で学んだ内容を、`template/web-basic` に **再実装** します。
- 課題は **commit / push** を行い、宿題として提出します。
- コミットメッセージには Issue 番号を含めます（例: `#3 add counter`）。
