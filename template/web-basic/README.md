# web-basic テンプレート

このテンプレートは、HTML/CSS/JavaScript の最小構成で動作確認を行うための土台です。講座で学んだ内容を小さく再実装し、GitHub提出の練習に使います。VS Code と Live Server を使うと、初学者でも手順を再現しやすくなります。

## 使い方（VS Code + Live Server）

1. `template/web-basic` を VS Code で開く
2. `index.html` を右クリックし、**Open with Live Server** を選択する
3. ブラウザで画面を開き、ボタンを押してカウンタが増えることを確認する

## ブラウザ確認手順

1. 見出し「JavaScript Entry Template」が表示される
2. `+1` ボタンをクリックする
3. `count` の表示が `0 → 1 → 2 ...` と増える

## 宿題提出手順（Issue → Branch → PR）

1. Issue を作成し、やることを明確にする
2. Issue 番号付きでブランチを作成して実装する
3. commit / push して Pull Request を作成する
